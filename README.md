The Toe Show is a open-source project anyone can fork remake and you might as well add GUI and change. Hope you guys enjoy it!
NOTE: You're OS may need nano for directory change, this was made on Twister OS so some other OSes may need modification of the .sh script, finally your system needs termnial, Bash/Shell, and Python 3
For systems with chmod +x & nano (if needs modification)
chmod +x /directory/to/TheToeShow.py
nano /directory/to/TheToeshow
In nano do:
#!/bin/bash
python3 /home/pi/mu_code/Saves/TheToeShow.py
Save & exit
chmod +x /directory/to/TheShow
To run in terminal do cd /directory/to/TheToeShow or click it in the file system.
